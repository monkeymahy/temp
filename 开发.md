## miniforge环境

从[miniforge github](https://github.com/conda-forge/miniforge/releases/tag/25.9.1-0)网站下载安装包，当前最新版本为Release 25.9.1。

### 构造conda虚拟环境(除了swanlab，可以完全conda方式安装)

```python
conda create -n pyocc python=3.10
conda install anaconda::numpy==1.26.4
# Torch2.2.1 + CUDA 12.1
conda install pytorch==2.2.1 torchvision==0.17.1 torchaudio==2.2.1 pytorch-cuda=12.1 -c pytorch -c nvidia
conda install conda-forge::torch-ema
conda install conda-forge::torchmetrics
conda install conda-forge::tqdm
conda install -c conda-forge pythonocc-core=7.9.0
conda install conda-forge::timm
conda install -c dglteam/label/cu121 dgl
conda install anaconda::pandas
conda install lightning -c conda-forge
pip install swanlab


# 设置清华源
pip config set global.index-url https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple
# 安装<2版本的numpy
pip install numpy==1.26.4
# Torch2.2.1 + CUDA 12.1
pip install torch==2.2.1 torchvision==0.17.1 torchaudio==2.2.1 --index-url https://download.pytorch.org/whl/cu121
pip install torchdata==0.7.1
# 安装DGL库
pip install dgl -f https://data.dgl.ai/wheels/cu124/repo.html
pip install torch-ema swanlab torchmetrics tqdm

# conda安装pythonocc-core
conda install -c conda-forge pythonocc-core=7.9.0

# 保存python环境
pip list --format=freeze > requirements.txt
```
